# Client logos

Upload the client logo files into this folder (this same
`site/public/clients/` directory).

- Any web format is fine: PNG, JPG, or SVG. PNG or SVG preferred.
- Name them simply, e.g. `adamas-films.png`, `aidbylaw.png`,
  `aqua-marine.png`, `hamprigo.png`, `kopa.png`, `tiksh.png`,
  `linear-grid.png`, `lumagreens.png`, `momento.png`, `ramat.png`,
  `tikasaheb.png`, `hettees.png`. Exact names do not matter - just keep
  them readable.
- Once the files are here, they will be wired into a "Trusted by" strip
  on the homepage and a clients section on the Work page (logos only, no
  names). White backgrounds will be removed so they sit cleanly.

This README is only a placeholder so the folder exists on GitHub; it can
be deleted once the logos are added.
